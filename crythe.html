CrGr: A CRYTHE User Program for Sequencing Music by Designing Simple Automata

CrGr is a moderately small User Program included with CRYTHE that's designed for creating MIDI sequencers
in real time, driven by simple automata comprised of various generators and effects, laid out in a 2D grid using the keyboard.
CrGr comes with rys, a built-in scripting language which aids users with entry and modification of values
in arbitrary logical clusters.
